NIR tools

This is a user-friendly and lightweight web-based suite for training and predicting near-infrared (NIR) spectra. This suite is based on our python package pynir,which was developed to handle a variety of problems in NIR calibrations and includes many more rich features.

Regression

This module contains regression with partial least quares (PLS), and other functions are coming soon...

Classification

This module contains classification with partial least quares discrimination analysis (PLS-DA), and other functions are coming soon...

Data Preprocessing

This module realizes data preprocessing of NIR spectra with continuous wavelet transform (CWT), standard normal variate (SNV), multiplicative scatter correction (MSC) and Savitzky-Golay (SG) filter based smooth and derivate, and other functions are coming soon...

Outlier Dection

This module is designed for outlier detection of NIR spectra.

Feature Selection

This module is designed for feature selection of NIR spectra. Currently, it offers algorithms such as VIP, RT and VC, with support for additional algorithms planned for the near future.

Calibration transfer

This module realizes calibration transfer/enhancement of NIR spectra with parameter free calibration enhanccement(PFCE) framework, and other functions are coming soon...

Details of the PFCE see the reference 1 and 2.

Utils

This is a collection of functions of being useful for training a more accurate model.
